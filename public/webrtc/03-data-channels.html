💬 WebRTC Example 3: Data Channels

What this demonstrates: Using RTCDataChannel to send arbitrary data between peers. Data channels can transfer text, binary data, files, or any serializable data peer-to-peer without going through a server. Perfect for chat, gaming, or collaborative apps.
Key features of Data Channels:
  • Reliable or unreliable: Choose TCP-like reliability or UDP-like speed
  • Ordered or unordered: Messages can arrive in any order for speed
  • Binary or text: Send strings, ArrayBuffers, or Blobs
  • Low latency: Direct peer-to-peer communication
